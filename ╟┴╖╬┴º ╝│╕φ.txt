<주의 사항>
1. 컴퓨터 셋팅 시 즐겨찾기를 보여주기 위하여 미리 몇 번 실행하여 레지스트리를 등록하도록 해야 한다.
	=>  처음 실행시 동적으로 자동 셋팅된다. 하지만, 즐겨찾기 몇 개 추가해줄 것...
2. 컴퓨터 셋팅 시 미리 알람을 셋팅하여 준다.
	=>  처음 실행 시 알람 기본 셋팅이 동적으로 생성된다. 하지만 음악 파일 설정이 안 되어있으므로 알람 음악 파일 설정 해주기.
3. 동영상과 사운드 미디어 파일을 재생할 수 있으므로 스피커를 달아놓는다.
4. 컴퓨터 셋팅 시 동영상 파일 재생 예제가 있어야 하므로 AVI file 이나 MPEG 파일 암 거나 다운받아 놓는다.

프로그램 명 : 통합 웹 브라우저

제작 툴 : Visaul C++ (COM, MFC, API)

제작자 : 신 태규(컴퓨터 공학과 01)

소개글

다양한 멀티미디어 기능을 포함한 브라우저를 제작하였다.

통합 멀티미디어 솔루션을 가지고 있는 통합 브라우저이다.

다양한 기능들을 포함하고 있다..

1. 즐겨찾기 관리 기능 (레지스트리 이용)
 => 메뉴에 있는 Add Favorites 항목을 클릭하여 현재 띄워진 웹사이트를 즐겨찾기에 등록할 수 있다. (레지스트리에 등록)
 => 등록된 즐겨찾기는 메뉴의 '즐겨찾기' 항목에 등록된다.
 => 그리고 툴바 밑의 링크바에서 '즐겨찾기' 옆의 조그만 화살표를 클릭하면 똑같이 즐겨찾기가 뜬다.
 => 즐겨찾기 중에 마음에 드는 항목을 눌러서 해당 주소로 여행할 수 있다.
 => 프로그램이 끝나도 즐겨찾기는 살아 있다.

2. 방문한 URL 관리 기능 (레지스트리 이용)
 => 방문했던 URL 들은 '기능' 메뉴를 클릭하면 기능 메뉴 아래로 방문했던 URL 들이 추가되어 나타난다.
 => 추가된 URL 들을 클릭하여 다시 여행할 수 있다.

3. AVI , MPEG , WAV 등 다양한 미디어 파일 재생 가능 ('사사미' 버튼 클릭)
 => '사시미' 툴바 버튼을 클릭하면 다이얼로그가 뜨는데 'Load File' 을 클릭하여 원하는 파일 선택 후 PLAY 버튼을 클릭하면 플레이가 된다.
 => 플레이 중간에 PAUSE 와 END 가 가능하며, 슬라이더바를 드래그 하여 동영상을 중간부터 보거나 하는 기능은 없다. 주의할 것 ! ㅡ_ㅡㆀ

	(MPEG, AVI ,WAV 는 로드되는 것으로 실험해보았다. 그 외에 많은 형식의 동영상-사운드 파일을 돌릴 수 있다.)

추가하려 했지만 귀찮았다 =_=;

4. FTP Client Program 이 내장되어 있음. ('올FTP' 버튼을 클릭)
 => '올 FTP' 버튼을 클릭하면 다이얼로그 박스를 통해 FTP 에 접속할 수 있다.
 => 접속된 후에 위쪽에 나타나는 리스트가 Server이고 , 아래 쪽에 나타나는 리스트가 Local 에 있는 파일들이다.
 => 접속 후 ListBox 를 클릭시에 , 서버는 로컬로, 로컬은 서버로 자동 업로드-다운로드가 된다.
 => 컨트롤 끼리 드래그 & 드롭이 가능하다. 즉, 리스트 서 다른 리스트로 드래그 하면 자동적으로 업-다운로드가 된다.
 => 리스트 에 아무 파일이나 클릭 후 옆의 버튼을 눌러도 된다.
 => 리스트 에서 직접 파일을 지우거나 파일 이름을 바꿀 수도 있다. 서버의 파일도 설정 변경 가능...
      ( Ctrl + R 입력하면 다이얼로그가 뜨는데, 새로운 파일 이름을 입력하면 해당 이름으로 이름이 바뀐다.
        DEL 키를 입력하면 현재 포커스가 있는 파일이 삭제가 된다. 물론 정말로 삭제할 것인지 여부를 한 번 더 물어본다.)

5. 동적 알람 기능이 내장되어 있어서, 사용자가 원하는 대로 알람을 맞추거나 해제 가능하다.
 => 브라우저 '옵션' 메뉴를 클릭하면 알람에 대하여 자기가 조정할 수 있다.
      ( 알람 시간 조정 , 알람 취소/설정 , 알람 사운드 설정 )

6. 뒤로/앞으로/홈으로/중지/새로고침 등의 브라우져 기본 옵션을 지원한다.

7. '브라우저 숨기기' 기능
 => '브라우저 숨기기' 툴바 를 클릭하면 브라우저가 사라진다. 프로그램이 끝난 게 아니고, '아래 오른쪽' 에 트레이 아이콘에 보면 브라우져 아이콘이 뜬다.

8. '전체 화면' 기능
 => 'FULLSCREEN' 메뉴나 툴바를 클릭시 브라우져가 전체화면으로 떠서 더 큰 브라우저를 경험할 수 있다.
